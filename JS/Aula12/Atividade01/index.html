Aula 12 - Introdução ao Java Script (JS)

Atividade 01

  1. Crie um script que peça ao usuário o nome, número e cidade. Imprima essas informações no console. Mostre como um alerta a cidade informada pelo usuário. Altere o número para 1000 e exiba no console.
  • use console.log() para exibir no terminal do navegador
  • use alert() para exibir no navegador
  • Use prompt() para obter informação do usuário
  • Let para todas as variáveis
